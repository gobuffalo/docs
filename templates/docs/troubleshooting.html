Because the templates are in the first, original `$GOPATH`, the installer does not find them, and subsequently generates an incomplete application. To fix this, use just one `$GOPATH`. * Using a single `$GOPATH`. If you aren't using multiple `$GOPATH`s and are still seeing this issue, it is most likely caused by a bad installation.
